Quadratic Programming Control Allocation Toolbox for Matlab
Version 1.2.1 (August 25, 2004)

 

ABOUT THE TOOLBOX

The Quadratic Programming Control Allocation Toolbox (QCAT) provides Matlab implementations of a number of algorithms for control allocation that can be found in the literature. All functions have been implemented and optimized for speed by Ola Härkegård except the interior point solver contributed by John Petersen.

The toolbox can be downloaded here. Please send an e-mail if you have

  • questions or comments about the toolbox;
  • found a bug in the code;
  • developed your own algorithm, that you wish to incorporate into the toolbox to be able to compare the computational complexity or other properties of the solver to the existing algorithms.
