WPINV - Compute weighted pseudoinverse.

SYNOPSIS

function [G,F] = wpinv(A,W)

DESCRIPTION 

 WPINV - Compute weighted pseudoinverse.
 
  [G,F] = wpinv(A,W)
 
 Computes the optimal solution x = Gy + Fx0 to the least squares
 problem
 
  min ||W(x-x0)||  subj. to  Ax = y
   x

 See also PINV.
